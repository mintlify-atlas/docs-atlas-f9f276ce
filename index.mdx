## Built With Modern Technology

OSINT Hub is built on a solid foundation of proven technologies:

- **Python 3.12** and **Django 5.2** for robust backend
- **Celery 5.6** with **Redis** for async task processing
- **Bootstrap 5** for responsive, accessible UI
- **Holehe** for email search across platforms
- **Sherlock** for username search on 300+ websites
- **ExifTool** for comprehensive metadata extraction
- **WhiteNoise** for efficient static file serving
